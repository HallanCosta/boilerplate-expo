# :iphone: How to Use
1. Clone the repository
```sh
 git clone git@github.com:HallanCosta/boilerplate-expo-atomic-design.git
```

2. Install the dependencies and Execute boilerplate</p> 
```sh
# Install.
$ yarn install # ou npm install

# Execute.
$ yarn start # ou npm start
```
